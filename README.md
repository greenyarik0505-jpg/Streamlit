# Streamlit Quiz

Небольшое Streamlit-приложение с викториной на знание разных тем. Пользователь выбирает ответы в радио-кнопках, а приложение сразу показывает, верно ли выбран вариант.

## Описание

Проект представляет собой простой учебный веб-интерфейс на Python и Streamlit. Внутри несколько вопросов с проверкой ответов и визуальной обратной связью через `st.success()` и `st.error()`.

Это хороший стартовый пример для:

- учебных викторин;
- мини-опросов;
- проверки знаний по теме;
- быстрого прототипа на Streamlit;
- простого интерактивного приложения без сложной логики.

## Что умеет приложение

- показывает заголовок страницы;
- выводит несколько вопросов подряд;
- даёт выбор из вариантов ответа;
- моментально отмечает правильный или неправильный ответ;
- не требует базы данных и дополнительных экранов;
- запускается как обычное Streamlit-приложение.

## Содержимое викторины

В текущей версии есть вопросы про:

- футбол;
- еду;
- интернет-мемы и brainrot-тематику;
- исторические государства;
- физику.

Каждый вопрос проверяется отдельно, поэтому пользователь получает мгновенный результат после выбора варианта.

## Как запустить

### 1. Установить зависимости

Если Streamlit ещё не установлен:

```bash
pip install streamlit
```

### 2. Запустить приложение

Из папки проекта:

```bash
streamlit run main.py
```

### 3. Открыть в браузере

После запуска Streamlit покажет локальный адрес, обычно:

```text
http://localhost:8501
```

Открой его в браузере и проходи викторину.

## Как работает логика

В `main.py` используется одинаковый шаблон для каждого вопроса:

1. `st.radio()` показывает варианты ответа.
2. Условие `if` сравнивает выбранный вариант с правильным.
3. `st.success()` показывает правильный ответ.
4. `st.error()` показывает ошибку, если выбран неверный вариант.

Такой подход очень удобен для небольших обучающих приложений и легко расширяется новыми вопросами.
